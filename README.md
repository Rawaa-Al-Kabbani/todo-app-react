# Todo calendar

Todo calendar is a todo/calendar website where you can sign up, add todos to your own personal calendar, tag them, and eventually mark them as done. The back-end is written in Node.js together with Express.js and MySQL for the database. The front-end is written in React.js and Material-Ui.

## Running the website locally

To run the website locally first clone this repository, configure the database and then run `npm install` and `npm start`. The website is served on [http://localhost:3000](http://localhost:3000) by default.
